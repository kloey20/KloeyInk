This is the first version of the Daily Verses plugin

It uses https://beta.ourmanna.com and API to get the verses


Changes:
- Added dynamic fonts
The fonts are increased when the text is shorter but has a limit of 90 pts. Vice versa, when the text is longer, it is reduced to a smaller font.
